Setup Environments
--------------------

1. Clone from `GitHub <https://github.com/ytakzk/Gradual-Assemblies>`_

2. Open Rhinoceros

3. EditPythonScript -> Tools -> Options

.. image:: https://github.com/KEERTHANAUDAY/timber_grammar/blob/master/docs/source/_static/Install_1.PNG


4. | Add a path to `UR_Control`
   | It depends where you locate this repository but it must be like "`C:\\Users\\{your_name}\\GitHub\\timber_grammar\\src`"

.. image:: https://github.com/KEERTHANAUDAY/timber_grammar/blob/master/docs/source/_static/Install_2.PNG
5. Restart Rhinoceros
